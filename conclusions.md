# Pipeline Test Conclusions

**Input:** ~580 токенов, 215 уникальных лемм
**Level:** B1
**Candidates found:** 32

### Проблемы

1. **Proper nouns не фильтруются** — `morty`, `rick` попадают в кандидаты, потому что spaCy не распознаёт их как имена собственные в нестандартных формах (`Rickless`, `Mortys`, `Mortytown`). Мусор в кандидатах.

2. **Вульгарные/разговорные слова** — `ass`, `em` проходят фильтр (cefrpy присваивает им B2+, они не stop words). Нужен отдельный фильтр или возможность массово помечать как skip.

3. **Слишком частые слова** — `cause` (Zipf 5.3), `cast` (Zipf 4.8), `campaign` (Zipf 5.0) попадают только из-за CEFR уровня, но они слишком распространены чтобы их учить. Возможно стоит добавить верхнюю границу Zipf.

4. **Сверхредкие слова** — `defame`, `coax`, `outnumber` (Zipf < 3.0) — вне sweet spot, встречаются слишком редко для практической пользы.

### Выводы для будущих улучшений

- Улучшить детекцию proper nouns для нестандартных имён (см. improvements.md — phrasal verbs)
- Рассмотреть верхнюю границу Zipf (например 5.0) для фильтрации слишком частых слов
- Рассмотреть стоп-лист вульгарных/разговорных слов или UI для быстрого массового skip
